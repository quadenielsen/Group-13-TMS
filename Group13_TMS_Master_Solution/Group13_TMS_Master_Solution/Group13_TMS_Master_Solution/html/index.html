Shapes Project

Program Introduction

The myShape application is meant to show how parent classes and child classes are used to create inheritance relationships between classes. It also allows you to play with shapes! And it serves as a example of DOxygen commenting style.

Major myShapes features to note are:

  • You can create circles and/or squares
  • You can give your shapes different colours
  • You can give your shapes different sizes
