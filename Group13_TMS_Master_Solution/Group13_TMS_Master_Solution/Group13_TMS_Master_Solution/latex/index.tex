\hypertarget{index_intro}{}\doxysection{Program Introduction}\label{index_intro}
The {\bfseries{my\+Shape}} application is meant to show how {\itshape parent classes} and {\itshape child classes} are used to create inheritance relationships between classes. It also allows you to play with shapes! And it serves as a example of {\itshape DOxygen commenting style}.

Major my\+Shapes features to note are\+:


\begin{DoxyItemize}
\item You can create circles and/or squares
\item You can give your shapes different colours
\item You can give your shapes different sizes
\end{DoxyItemize}

Not sure what a shape is? Here is some information on shapes\+:
\begin{DoxyItemize}
\item \href{https://en.wikipedia.org/wiki/Shape}{\texttt{ Shapes Wikipedia Article}}
\end{DoxyItemize}
